Tracy


  1. Tracy is a software prototype designed to produce an hybrid 2D & 3D animation rendering. I made the GUI as well as the core.

  2. How does it work?

    1. Import a 3D animation buffer

    2. Draw in 2D over the 3D on the screen

    3. Launch drawings interpolation between key frames.

    4. Export the 2D animation buffer for compositing.
