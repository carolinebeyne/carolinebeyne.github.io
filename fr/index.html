Tracy


  1. Tracy est un prototype de logiciel conçu pour produire un rendu d'animation hybride 2D & 3D. J'ai réalisé l'interface ainsi que l'architecture logicielle.

  2. Comment ça marche ?

    1. Importer un cache d'animation 3D

    2. Dessiner en 2D sur la 3D à l'écran

    3. Lancer l'interpolation des dessins entre les images-clés

    4. Exporter le cache d'animation 2D pour le compositing

Reconnaissance de signatures


  1. Dans le cadre de notre cours de Classification de Données, avec 3 camarades de classe

  2. Nous avons utilisé un papier scientifique à l'état de l'art sur le sujet pour le développement

  3. La base de données comporte des signatures originales et des contrefaçons que notre programme, fait en C++, traite en 2 étapes

    1. D'abord, les données d'apprentissage avec seulement des signatures originales

    2. Ensuite, les données de test

    3. Pour chacune des signatures de test, l'algorithme décide si elle est vraie ou fausse


    http://bit.ly/By0ute-TRPA
